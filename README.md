# David Bocaranda — Portfolio

Portfolio website for David Bocaranda, WordPress Developer & SEO Specialist.

## Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Syne, Space Mono, Inter)
- No frameworks, no dependencies — pure static site

## Deploy
This site is ready to deploy on Vercel, Netlify, or any static hosting.

1. Push to GitHub
2. Import repo in Vercel
3. Deploy — no build step required

## Structure
```
portfolio/

├── style.css      # All styles
├── script.js      # Interactions & animations
└── README.md
```
